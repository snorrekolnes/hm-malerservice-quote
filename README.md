# HM Malerservice AS - pristilbud

Internt, klientdrevet Next.js-verktøy for å lage norske pristilbud og skrive dem ut som A4/PDF.

## Start lokalt

```bash
npm install
npm run dev
```

Åpne deretter `http://localhost:3000`.

## Bruk

Fyll inn prosjekt, kunde, dato og tilbudsposter. `Sum eks mva`, `Mva (25%)` og `Sum inkl mva` beregnes automatisk. Trykk `Generer PDF` og velg `Lagre som PDF` i utskriftsdialogen.

Alt kjører i nettleseren. Det er ingen innlogging, database eller serverlogikk.
